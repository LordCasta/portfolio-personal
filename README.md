## Tecnologías utilizadas

### Frontend
- HTML
- CSS (Tailwind CSS + estilos personalizados)
- JavaScript

### Tecnologías mostradas en el portafolio
- **Frontend:** HTML, CSS, JavaScript, React JS, Tailwind CSS  
- **Backend:** PHP, Laravel, MySQL, SQL Server, MongoDB  
- **Pruebas:** Java, Selenium, Cucumber

## Cómo ejecutar el proyecto

Como es un sitio estático, no requiere instalación de dependencias.

1. Clona el repositorio:
   ```bash
   git clone https://github.com/lordcasta/portfolio-personal.git
   ```

2. Entra a la carpeta del proyecto:
   ```bash
   cd portfolio-personal
   ```

3. Abre el archivo `src/index.html` en tu navegador.

> Recomendado en VS Code: usar la extensión **Live Server** para visualizar cambios en tiempo real.

## Personalización

Puedes editar fácilmente:

- `src/index.html`:
  - Nombre
  - Descripción personal
  - Tecnologías
  - Proyectos
  - Experiencia laboral
  - Contacto

- `src/styles/main.css`:
  - Colores
  - Tamaños
  - Animaciones
  - Estilos del hero y tarjetas

- `src/scripts/main.js`:
  - Animaciones de scroll
  - Interacción del fondo del hero
  - Movimiento de logos tecnológicos
